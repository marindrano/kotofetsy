# Crawler

Python crawler for Malagasy news sources, built on [Scrapling](https://github.com/D4Vinci/Scrapling). See [`docs/architecture/crawler.md`](../../docs/architecture/crawler.md) for responsibilities and open questions.

## Setup

Managed with [uv](https://docs.astral.sh/uv/). Install deps:

```sh
uv sync
```

## Run

```sh
uv run main.py
```
